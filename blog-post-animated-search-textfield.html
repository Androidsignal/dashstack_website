Creating Beautiful Animated Search with animated_search_textfield

Learn how to implement smooth and elegant search animations in your Flutter applications with our open-source package.

DashStack Team Flutter Package Maintainers
Feb 9, 2026 6 min read

User experience is everything in modern app design. A well-crafted search interface can make the difference between an ordinary app and one that users love. The animated_search_textfield package from DashStack makes it easy to create beautiful, animated search experiences in Flutter.

What is animated_search_textfield?

This package provides a customizable and animated search text field widget that expands smoothly when tapped and collapses with elegant animations. It's the perfect solution for creating professional search UIs without complex custom widget development.

Key Features

  • Smooth Animations: Expand/collapse with fluid transitions
  • Customizable Colors: Adapt to your app's design system
  • Custom Icons: Use any icon you prefer
  • Custom TextField: Pass your own TextField for full control
  • Custom FAB Support: Replace the default floating action button
  • Lightweight: Minimal dependencies

Installation

Add to your pubspec.yaml:
dependencies:
  animated_search_textfield: ^0.0.2

Basic Usage

The simplest implementation requires just a few lines of code:

import 'package:animated_search_textfield/animated_search_textfield.dart';

@override
Widget build(BuildContext context) {
  return Scaffold(
    body: Column(
      children: [
        AnimatedSearchTextField(
          expandedWidth: 300,
          searchTextField: TextField(
            decoration: InputDecoration(
              border: InputBorder.none,
              hintText: "Search here...",
              contentPadding: const EdgeInsets.symmetric(horizontal: 16),
            ),
            style: const TextStyle(color: Colors.black, fontSize: 16),
          ),
          iconSize: 30,
          iconBackgroundColor: Colors.deepOrange,
          iconColor: Colors.white,
        ),
      ],
    ),
  );
}

Customization Options

The package offers extensive customization:

expandedWidth Width when expanded (default: 300)
duration Animation duration (default: 600ms)
iconBackgroundColor FAB background color
iconColor Icon color (default: white)
searchIcon Icon when collapsed (default: Icons.search)
backIcon Icon when expanded (default: Icons.arrow_back)

Advanced Customization Example

AnimatedSearchTextField(
  expandedWidth: 300,
  duration: const Duration(milliseconds: 400),
  searchIcon: Icons.search_outlined,
  backIcon: Icons.close,
  iconColor: Colors.white,
  iconBackgroundColor: Colors.blueAccent,
  searchTextFieldColor: Colors.grey[100],
  searchTextField: TextField(
    onChanged: (value) {
      // Handle search input
    },
    decoration: InputDecoration(
      border: InputBorder.none,
      hintText: "Search products...",
      hintStyle: TextStyle(color: Colors.grey[400]),
      contentPadding: const EdgeInsets.symmetric(horizontal: 16),
    ),
  ),
  floatingActionButton: FloatingActionButton.large(
    backgroundColor: Colors.blueAccent,
    child: const Icon(Icons.search),
    onPressed: () {},
  ),
)

How It Works

  • Collapsed State: Shows as a circular FAB with search icon
  • Tap Gesture: Smoothly expands into a horizontal search bar
  • Expanded State: Displays custom TextField for user input
  • Auto-collapse: Back button or external trigger collapses the search bar
  • Text Clearing: Automatically clears input when collapsing

Real-World Use Cases

This package is ideal for:

  • E-commerce apps with product search
  • Content discovery applications
  • Social media feeds with filtering
  • Music and video streaming apps
  • Task management and todo apps
  • Any app where space-efficient search is needed

Pro Tips

Combine this package with proper debouncing for search queries to improve performance and reduce API calls in real-time search scenarios.
  • Use with FutureBuilder for async search results
  • Implement debouncing to optimize search API calls
  • Match your app's color scheme for a cohesive design
  • Test animation performance on various devices

Conclusion

The animated_search_textfield package is a powerful tool for creating professional, animated search experiences with minimal effort. Whether you're building an e-commerce app or a content discovery platform, this package can help you create a polished, modern search interface that users will love.
