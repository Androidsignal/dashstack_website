PriceText: Multi-Currency Formatting Made Easy

Master the art of displaying prices across multiple currencies with proper localization and formatting in Flutter apps.

DashStack Team Flutter Package Maintainers
Feb 7, 2026 8 min read

Building global applications that support multiple currencies can be challenging. Handling currency formatting, displaying symbols correctly, and maintaining localization across different regions requires careful planning. The price_text package from DashStack simplifies this complexity into a single, easy-to-use widget.

What is PriceText?

PriceText is a powerful Flutter package that handles currency formatting and display across 13+ currencies with automatic localization support. It combines currency symbols, flags, codes, and customizable formatting into one elegant widget.

Supported Currencies

The package supports the following currencies out of the box:

  • USD (United States Dollar)
  • EUR (Euro)
  • GBP (British Pound)
  • INR (Indian Rupee)
  • JPY (Japanese Yen)
  • AUD (Australian Dollar)
  • CAD (Canadian Dollar)
  • CNY (Chinese Yuan)
  • SGD (Singapore Dollar)
  • NZD (New Zealand Dollar)
  • MXN (Mexican Peso)
  • ZAR (South African Rand)
  • TRY (Turkish Lira)

Installation

Add to your pubspec.yaml:
dependencies:
  price_text: ^0.0.7

Basic Usage

The simplest implementation requires just a couple of parameters:

import 'package:price_text/price_text.dart';

PriceText(
  currencyType: CurrencyType.USD,
  amount: 1000.50,
)

This displays: $1,000.50

Key Features

  • Multiple Currencies: Support for 13+ currencies worldwide
  • Auto Formatting: Handles decimal places and thousand separators
  • Currency Symbols & Codes: Display $ £ ₹ and currency codes
  • Country Flags: Show flag icons for each currency
  • Color Coding: Automatic colors for positive (green), negative (red), and zero values
  • Custom Styling: Full control over text appearance
  • Localization: Regional formatting using intl package

Advanced Example: E-Commerce

PriceText(
  currencyType: CurrencyType.USD,
  amount: 12345.67,
  showFlag: true,                    // Show country flag
  showCurrencyCode: true,            // Show "USD" text
  flagSpacing: 10,                   // Space between flag & code
  flagAlignment: AlignmentOption.left, // Flag on left side
  locale: "en_US",                   // US formatting
  formatterPattern: '#,##0.00',      // Custom number format
  amountColorStyle: (amount) {
    if (amount > 0) return Colors.green;
    if (amount < 0) return Colors.red;
    return Colors.grey;
  },
)

Customization Options

The package provides extensive customization:

showFlag Display country flag (true/false)
showCurrencyCode Display currency code like "USD"
hideCurrencySymbol Hide $ € ₹ symbols
flagAlignment Position flag left or right
locale Regional formatting (en_US, en_IN, etc.)
formatterPattern Custom number format like #,##0.00
customFlagWidget Use your own flag image instead of default

Handling Positive and Negative Values

PriceText automatically applies different colors based on amount values: 

// Positive amount shows green
PriceText(
  currencyType: CurrencyType.USD,
  amount: 1000,
)

// Negative amount shows red
PriceText(
  currencyType: CurrencyType.USD,
  amount: -1000,
)

// Zero amount shows grey
PriceText(
  currencyType: CurrencyType.USD,
  amount: 0,
)

Localized Formatting

Different regions format currencies differently. PriceText handles this automatically:

  • en_US: $1,234.56
  • en_IN: ₹12,34,567.89
  • de_DE: 1.234,56 €
  • fr_FR: 1 234,56 €

Real-World Applications

Perfect for:

  • E-commerce and shopping apps
  • Banking and financial applications
  • Cryptocurrency and trading apps
  • Travel and booking platforms
  • Expense tracking applications
  • Multi-currency wallets

Pro Tips

Always use the locale property when building apps for international users. This ensures that number formatting follows regional conventions and improves user experience.
  • Combine with StreamBuilder for real-time price updates
  • Use custom color logic for financial applications
  • Leverage custom flags for branded experiences
  • Cache currency rates to minimize API calls

Conclusion

The price_text package eliminates the headache of currency formatting in Flutter apps. Whether you're building a simple online store or a complex financial application, PriceText provides the tools you need to display prices professionally across multiple currencies and locales. Download it today and simplify your currency handling!
