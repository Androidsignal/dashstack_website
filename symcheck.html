Challenges & Solutions

Building a cutting-edge medical vitals scanning application required overcoming immense hurdles in camera hardware access, image processing, and HIPAA compliance.

Live Photoplethysmography

Extracting heart rate from facial micro-color changes requires massive computational power.

We wrote custom C++ computer vision algorithms and executed them via Flutter FFI directly on the device's neural engine.

Cross-Device Camera Parity

Different Android manufacturers apply wildly different default post-processing to camera feeds.

Bypassed standard camera APIs to pull raw, uncompressed frames directly from the hardware sensors for consistent algorithmic accuracy.

Medical Data Security

Handling raw vital data required uncompromising adherence to healthcare privacy laws.

Implemented AES-256 encryption for all temporary files and ensured zero persistent local storage of any medical data.

Foolproof User Guidance

Users struggled to position their faces correctly for the algorithmic scan.

Engineered a dynamic augmented reality (AR) overlay mask that turns green only when lighting and positioning are mathematically perfect.
